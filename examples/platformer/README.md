# Red Panda Ridge

A small Vite + React 18 + PixiJS v8 + TypeScript platformer — built and
verified end-to-end by persona agents dogfooding canon, one agent per
canon voice. Three levels, pinecone enemies, hearts, moving platforms,
a React HUD with pause/level-select/records, and a red panda who just
wants to find their squirrel friend.

## Play

```bash
cd examples/platformer
bun install
bun run dev        # http://localhost:5173
# arrows/WASD + space to jump, Escape/P pause, R restart
```

`bun run build && bun run preview` for the production build. The
package is standalone — not part of the repo's Bun workspace.

## How it was built (the canon dogfood)

Every persona authored REAL canon artifacts with the real CLI:

| Voice | What they did | canon surface |
|---|---|---|
| planning | wrote the v2 implementation plan (6 tasks, 59 steps, writing-plans format) | `docs/superpowers/plans/2026-07-14-red-panda-ridge-v2.md` → mirrored by `canon ingest plans` (dialect `superpowers`) |
| design | design one-pager + authored the 8 new behavior scenarios | `canon context` → `canon scenario new` × 8 → `canon fmt --check` (6 files, 0 violations) |
| dev | ported v1 → Vite/React/Pixi, implemented features against the 16-scenario corpus, flipped plan checkboxes | `canon query --kind scenario`, `canon retrieve --role dev` (pre-task guidance) |
| test | independently verified all 16 scenarios in-browser; fixed 2 defects it could prove | `canon review add` × 16 (attested, pinned to the verified sha) |
| review | code review; 8 confirmed ship-blockers filed as divergences; all fixed then resolved | `canon divergence stage/promote` × 8 → `resolve` × 6 ids |

The trust chain this demonstrates:

```
dev claims done   → plan checkboxes → canon ingest plans   (mirror, never trusted)
test proves done  → canon review add per scenario           (evidence)
review disagrees  → canon divergence stage/promote          (tracked, run_seq-ordered)
fix lands         → re-verify → attest → divergence resolve (burndown to zero)
                  → canon gate check: clean (0 violations)
```

Final state: 16/16 scenarios attested, divergence burndown
`open=0 resolved=6`, gate clean.

## Spec corpus

`specs/features/kind=feature/area=platformer/*.feature` — 16 tagged
Gherkin scenarios (movement ×4, session ×4, enemy ×2, levels ×2,
moving ×2, hud ×2). `canon inventory sync` materializes them into the
ledger index; they are the acceptance authority the personas worked
against.

## Assets

Pixel-art PNGs in `assets/`, generated by a sonic agent via image
generation and chroma-keyed at load: red panda hero, squirrel friend,
acorns, grumpy pinecone enemies, hearts, tiles, floating platforms,
and an autumn-sunset backdrop.
